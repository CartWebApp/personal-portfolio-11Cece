EECU Wise-Up Budget Calculator This project is to build a Financial Literacy Web App that helps students transition from high school to the workforce. Using real-world data from a Career API, the app should calculate exactly how much money a person actually takes home after the government takes its share, and whether their lifestyle choices lead to a "Surplus" or a "Shortfall". For this project, I programmed with HTML5, CSS3, Vanilla Javascript. I used Visual Studio Code as my code editor, Git/Github as my version control, and Netlify to host my project. This project was a leading point to establish the skills I’ve learned with all three different coding languages. I learned how to implement JavaScript libraries such as Chart.js to create an updating chart. Another concept I learned was DOM manipulation as I needed to take user-input from HTML tags to pass into Chart.js. I struggled with the initial building of the Chart as it would not show up the categories and styling my content with CSS. To resolve these issues, I looked back towards a JavaScript Chart practice assignment I was introduced. For my CSS, I looked up resources online and allowed feedback from my peers and instructors to help me resolve the issue. My challenge with chart.js ignited my habits of mind– being to investigate and communicate. These two were my habits of mind because I communicated with my peers and instructors when necessary for feedback on what I can do to improve better or discuss how to resolve issues within my code and I investigated for answers and resources to help myself understand.
